# DigitalOcean WordPress One Click Install

This bash script automates the creation of a DigitalOcean Droplet pre-configured with a one-click WordPress installation. It uses the doctl CLI to provision the server, making it quick to get a WordPress site up and running on DigitalOcean.

## Language

`bash`

## How to Run

1. Ensure you have doctl installed and authenticated.
2. Replace 'your-ssh-key-fingerprint' with your actual SSH key fingerprint.
3. Run the script: bash digitalocean_wordpress.sh
